BiBo Kernel Benchmark Pipeline

Isolated kernel-by-kernel testing with 4 mandatory rules

Baselines

  • • Baseline: Pure PyTorch eager (zero patches)
  • • Liger: RMSNorm + RoPE only
  • • Run once, all kernel tests compare against Baseline
  • • Speedup = baseline_median / triton_median

Kernel Isolation

  • • Each kernel tested on its OWN component
  • • Dense MLP: BiBoMLP module (not full model)
  • • MoE: BiBoFusedExperts module
  • • Conv: CausalConv1D + BiMoERouter
  • • No attention, no embedding, no LM head

4 Mandatory Rules

  • • Rule 1: Gradient equivalence vs original PyTorch
  • • Rule 2: NaN-free multi-pass (>=2 fwd+bwd)
  • • Rule 3: Three-phase timing (fwd/bwd/fwd+bwd)
  • • Rule 4: torch.profiler kernel breakdown
  • • If ANY rule fails: kernel is BROKEN
